# TutorMatch

TutorMatch is a learning marketplace connecting students, parents, tutors and coaching institutes.

## Categories

- School Tuition (UKG–12)
- JEE
- NEET
- CUET
- College Subjects
- Coding
- Spoken English

## Features

- Tutor Registration
- Parent Registration
- Requirement Posting
- Requirement Applications
- Tutor Verification
- Ratings & Reviews
- Admin Dashboard

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Shadcn UI
- PostgreSQL
- Prisma
- NextAuth
- Cloudinary
- Razorpay (Phase 2)
